# Portfolio Sirine Gherbaoui

Site portfolio statique construit a partir du CV, du portfolio PDF et des images effectivement presentes dans le portfolio.

## Ouvrir le site

Ouvrir `index.html` dans un navigateur.

## Structure

- `index.html` : accueil
- `projets.html` : etudes de cas
- `galerie.html` : peintures Sirine Siena
- `profil.html` : parcours et competences
- `contact.html` : candidature et documents
- `assets/images` : images extraites du portfolio et optimisees
- `assets/documents` : CV et portfolio PDF

## Deploiement

Le dossier peut etre deploye tel quel sur Vercel comme site statique.
